# Rock Paper Scissors ✊📄✂️  

I made this as my **second Python project** during class.  
It only took me around **30–40 minutes** to put together, so it’s not very optimized, but it works!  

## How the game works  
1. The computer randomly picks **Rock, Paper, or Scissors**.  
2. You type your choice.  
3. The program compares both and decides if you win, lose, or tie.  

## What I learned while making it  
- Using `if/elif/else` chains to handle multiple conditions.  
- Making the program feel interactive with user input.  
- That even if the code isn’t the best version of itself, it’s still okay.
